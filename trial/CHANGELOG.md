# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-04-08

### Added
- Initial project structure
- Server with Flask REST API and web dashboard
- Agent with Scapy packet capture and IPS enforcement
- Signature-based detection (SQLi, XSS, Directory Traversal)
- Port scan detection (SYN, FIN, NULL, XMAS)
- Rate anomaly detection
- Global blocklist management
- Cloud deployment support (Render, PythonAnywhere)
- Attack simulator scripts
